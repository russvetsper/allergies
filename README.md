# _Coin Combinations_

#### _User can input how many cent they need and we show Combinations _

#### By _**Yi Chiang & Russ Vetsper **_

## Description/Specs

| Behavior        |Output Example  |
| ------------- |-----:|
| $0.01  | 1 penny |
| $0.05   | 1 nickel|
| $0.10 | 1 dimes |
| $0.25 | 1 quarter |
| $0.04  |  4 penny |
| $0.08   |  1 nickel+4 penny  |
| $0.18  | 1 dimes +1 nickel +3 penny  |
| $0.34| 1 quarter +1 nickel +4 penny |
| $0.75 | 3 quarter|
